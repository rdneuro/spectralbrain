# Getting Started

New to SpectralBrain? Start here.

```{toctree}
:maxdepth: 1

installation
quickstart
```

1. **{doc}`installation`** — install the core library and pick the optional
   extras (`bayesian`, `viz`, `gpu`, `neuro`) you actually need.
2. **{doc}`quickstart`** — load a surface, compute a descriptor, render it, and
   choose the learning path that fits what you already know.
