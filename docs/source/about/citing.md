# Citing SpectralBrain

If SpectralBrain contributes to your work, please cite it. Citation metadata is
maintained in [`CITATION.cff`](https://github.com/rdneuro/spectralbrain/blob/main/CITATION.cff)
at the repository root, and GitHub exposes a ready-to-copy APA/BibTeX entry from
the **“Cite this repository”** button on the project page.

A companion software paper is in preparation for the
[Journal of Open Source Software (JOSS)](https://joss.theoj.org/); once
published, cite the JOSS DOI as the canonical reference.

```bibtex
@software{spectralbrain,
  author  = {Debona, Rodrigo},
  title   = {SpectralBrain: spectral shape analysis for brain structures},
  url      = {https://github.com/rdneuro/spectralbrain},
  license = {MIT}
}
```
